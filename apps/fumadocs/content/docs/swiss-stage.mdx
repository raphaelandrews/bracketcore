## Installation

```package-install
npx shadcn@latest add https://bracketcore.dev/r/swiss-stage.json
```

This component depends on [lucide-react](https://lucide.dev) for connector icons. It will be installed automatically via the shadcn CLI.

## Usage

```tsx
import { SwissStage, type SwissBracket } from "@bracketcore/registry";
```

```tsx
<SwissStage bracket={bracket} />
```

## Props

### `SwissStage`

| Prop           | Type                     | Description                                       |
| -------------- | ------------------------ | ------------------------------------------------- |
| `bracket`      | `SwissBracket`           | The Swiss bracket data with rounds and standings. |
| `className`    | `string`                 | Additional CSS classes for the root element.      |
| `onMatchClick` | `(match: Match) => void` | Callback fired when a match card is clicked.      |

### `SwissBracket`

```ts
interface SwissBracket {
  type: "swiss";
  rounds: SwissRound[];
  standings: SwissStanding[];
  winsToAdvance: number; // e.g. 3 for a 3-win advance format
  lossesToEliminate: number; // e.g. 3 for a 3-loss elimination format
}
```

### `SwissRound`

```ts
interface SwissRound {
  name: string;
  record?: string; // e.g. "1-0 / 0-1"
  matches: Match[];
}
```

### `SwissStanding`

```ts
interface SwissStanding {
  team: Team;
  wins: number;
  losses: number;
  tiebreaker?: number; // Buchholz or similar
  status?: "advancing" | "eliminated" | "pending";
}
```

## Theming

All colors use your shadcn theme variables directly (`--card`, `--border`, `--card-foreground`, `--muted-foreground`, `--primary`). No extra configuration needed.

## Structure

The component renders two sections:

1. **Rounds** — horizontally scrollable columns of match cards, each with a round name and optional win/loss record label
2. **Standings** — a table sorted by record with columns for rank, team, wins, losses, and status (Advanced / Eliminated)

Teams with `status: "advancing"` show a green "Advanced" badge. Teams with `status: "eliminated"` show a red "Eliminated" badge and dimmed text.
