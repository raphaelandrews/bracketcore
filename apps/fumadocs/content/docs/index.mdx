Bracketcore provides tournament bracket components you can drop into any React project. Install individual components via the shadcn CLI.

## Installation

### Via shadcn CLI

```bash tab="npm"
npx shadcn@latest add https://bracketcore.andrews.sh/r/match-card.json
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://bracketcore.andrews.sh/r/match-card.json
```

```bash tab="yarn"
npx shadcn@latest add https://bracketcore.andrews.sh/r/match-card.json
```

```bash tab="bun"
bunx shadcn@latest add https://bracketcore.andrews.sh/r/match-card.json
```
