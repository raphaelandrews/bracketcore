Bracketcore provides tournament bracket components you can drop into any React project. Install individual components via the shadcn CLI or use the `@bracketcore/registry` package directly.

## Installation

### shadcn registry

Add components one at a time:

```bash
npx shadcn add https://bracketcore.dev/r/single-elimination.json
```

### Package

Or install everything at once:

```bash
npm install @bracketcore/registry
```
