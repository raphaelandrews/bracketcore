## Installation

```bash
npx shadcn add https://bracketcore.dev/r/double-elimination.json
```

Or install the package directly:

```bash
npm install @bracketcore/registry
```

## Props

### `DoubleElimination`

| Prop | Type | Description |
| --- | --- | --- |
| `bracket` | `DoubleEliminationBracket` | The bracket data with upper, lower, and grand final. |
| `className` | `string` | Additional CSS classes for the root element. |
| `onMatchClick` | `(match: Match) => void` | Callback fired when a match card is clicked. |

### `DoubleEliminationBracket`

```ts
interface DoubleEliminationBracket {
  type: "double-elimination";
  upper: Round[];   // Upper bracket rounds
  lower: Round[];   // Lower bracket rounds
  grandFinal: Match;
}
```

### `Round`

```ts
interface Round {
  name: string;
  matches: Match[];
}
```

### `Match`

```ts
interface Match {
  id: string;
  round: number;
  position: number;
  bestOf?: number;
  scheduledAt?: string;
  status?: "upcoming" | "live" | "completed";
  teams: [MatchTeam, MatchTeam];
}
```

### `MatchTeam`

```ts
interface MatchTeam {
  team: Team | null;
  score: number;
  isWinner?: boolean;
}
```

### `Team`

```ts
interface Team {
  id: string;
  name: string;
  logo?: string;
  seed?: number;
}
```

## Structure

The component renders three sections stacked vertically:

1. **Upper Bracket** — labeled in green, uses `SingleElimination` internally
2. **Lower Bracket** — labeled in red, uses `SingleElimination` internally
3. **Grand Final** — a single `MatchCard`

Both upper and lower brackets inherit all sizing and theming CSS custom properties from `SingleElimination`.
