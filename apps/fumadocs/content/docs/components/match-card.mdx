## Installation

```package-install
npx shadcn@latest add https://bracketcore.dev/r/match-card.json
```

## Usage

```tsx
import { MatchCard, type Match } from "@bracketcore/registry";

const match: Match = {
  id: "demo",
  round: 0,
  position: 0,
  bestOf: 3,
  status: "completed",
  teams: [
    {
      team: { id: "spirit", name: "Spirit" },
      score: 2,
      isWinner: true,
    },
    {
      team: { id: "falcons", name: "Falcons" },
      score: 0,
      isWinner: false,
    },
  ],
};

export default function Page() {
  return <MatchCard match={match} />;
}
```

## Examples

### Live

A live match shows a destructive border, a pulsing live indicator in the header, and colored scores — green for the leading team, red for the trailing team.

<MatchCardLive />

### Upcoming

An upcoming match displays the scheduled time and relative day in the header. The format adapts automatically: "Today", "Tomorrow", weekday name (within 7 days), or "Month Day".

<MatchCardUpcoming />

### Scheduled (TBD teams)

When teams are not yet determined but the match is already scheduled, the header shows the date while both team slots display "TBD".

<MatchCardScheduled />

### TBD

When `team` is `null` and no schedule is set, the card shows italic "TBD" text with no header.

<MatchCardTBD />

## Props

### `MatchCard`

| Prop           | Type                     | Description                                                          |
| -------------- | ------------------------ | -------------------------------------------------------------------- |
| `match`        | `Match`                  | The match data to display.                                           |
| `className`    | `string`                 | Additional CSS classes for the root element.                         |
| `onMatchClick` | `(match: Match) => void` | Callback fired when the card is clicked. Adds hover styles when set. |

### `Match`

```ts
interface Match {
  id: string;
  round: number;
  position: number;
  bestOf?: number;
  scheduledAt?: Date | string;
  status?: "upcoming" | "live" | "completed";
  teams: [MatchTeam, MatchTeam];
}
```

### `MatchTeam`

```ts
interface MatchTeam {
  team: Team | null;
  score: number;
  isWinner?: boolean;
}
```

### `Team`

```ts
interface Team {
  id: string;
  name: string;
  logo?: string;
  seed?: number;
}
```

## Theming

Colors use your shadcn theme variables directly (`--card`, `--border`, `--card-foreground`, `--muted-foreground`, `--primary`). No extra configuration needed.

The card width can be customized via a CSS custom property:

| Property                | Default | Description |
| ----------------------- | ------- | ----------- |
| `--bracket-match-width` | `13rem` | Card width. |

```tsx
<div style={{ "--bracket-match-width": "16rem" }}>
  <MatchCard match={match} />
</div>
```
